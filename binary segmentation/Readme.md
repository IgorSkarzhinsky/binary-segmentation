# Семантическая сегментация аэроснимков (NVIDIA Drone Dataset)

## 📌 О проекте
Модель компьютерного зрения для **бина��ной семантической сегментации** кадров с дрона.  
Отделяет **землю** от **неба** на каждом пикселе изображения.

| Вход (кадр с дрона) | Предсказание модели |
|---------------------|---------------------|
| ![input](assets/example_input.png) | ![mask](assets/example_mask.png) |

## 🎯 Задача
- **Тип:** Бинарная семантическая сегментация (2 класса: земля/небо)
- **Вход:** RGB-изображение (256×256)
- **Выход:** Маска сегментации (0=земля, 1=небо)

## 🛠️ Стек технологий
- Python 3.10
- PyTorch
- Segmentation Models PyTorch (U-Net + ResNet34)
- OpenCV
- Google Colab (GPU T4)

## 📊 Датасет
- **Источник:** NVIDIA Aerial Drone Dataset
- **Размер:** ~4500 пар (изображение + маска)
- **Разбивка:** 80% train / 20% val

## 📈 Результаты
| Метрика | Значение |
|---------|----------|

| Точность на валидации | >95% |
